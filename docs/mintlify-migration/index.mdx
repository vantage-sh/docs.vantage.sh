## Additional and Community Resources

Vantage maintains a variety of resources to help make the lives of cloud practitioners easier. All resources below are free to use and open source, when possible.

<CardGroup cols={3}>
  <Card title="API Documentation" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-api.jpg" href="https://vantage.readme.io/reference/general" >
  The Vantage API makes it easy for you to programmatically access and read your cloud pricing data. Automate the management and creation of cost reporting resources, as well as programmatically create reports and filters. View the API docs to get started.
  </Card>
  <Card title="EC2Instances.info" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-ec2.jpg" href="http://instances.vantage.sh/" >
  Simplifies the complex task of pricing EC2 instances on AWS. With over a million pricing permutations due to instance types, regions, and availability zones, the site streamlines the process, enabling users to quickly find and compare the best resources for their needs.
  </Card>
  <Card title="AWS Cost Leaderboard" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-leaderboard.jpg" href="https://leaderboard.vantage.sh/" >
  A ranked list of AWS services that customers spend the most on. Vantage provides this anonymized and aggregated data to the community to shed light on cloud infrastructure costs.
  </Card>
  <Card title="Cloud Cost Handbook" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-handbook.jpg" href="http://handbook.vantage.sh/" >
  An open-source set of guides meant to help explain the complex pricing of public cloud infrastructure and service providers in easy-to-understand terms. This guide is hosted on GitHub and is open to anyone to contribute their knowledge to the community.
  </Card>
  <Card title="Slack Community" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-slack.jpg" href="https://vantage.sh/slack" >
  While the documentation here is meant to help answer any questions, you're also welcome to join our Slack community to add to the discussion and ask questions.
  </Card>
  <Card title="Cloud Cost Reports" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-cloud-cost-report.jpg" href="https://www.vantage.sh/cloud-cost-report" >
  Vantage uses anonymized data from thousands of connected accounts to snapshot industry spending patterns. View our quarterly Cloud Cost Reports to analyze trends over time.
  </Card>
  <Card title="cur.vantage.sh" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-cur.jpg" href="https://cur.vantage.sh/" >
  This tool helps you decode AWS Cost and Usage Report (CUR) billing codes for any AWS service. Search for specific services or billing codes to find detailed information about usage types and more.
  </Card>
  <Card title="Vantage Blog" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-blog.jpg" href="https://www.vantage.sh/blog/" >
  Read insightful articles related to cloud industry trends and news. Stay up to date on Vantage press releases and product news.
  </Card>
  <Card title="FinOps as Code" img="https://assets.vantage.sh/docs/index-cards/docs-site-card-finops-as-code.jpg" href="https://github.com/vantage-sh/finops-as-code" >
  This repository contains tutorials and demos for implementing FinOps as Code in your organization. Practice using the Vantage API and Terraform provider with step-by-step walkthrough tutorials.
  </Card>
</CardGroup>
